## 01. 我的叙事：从后端到 AI Builder

白天，我做高并发、稳定性、可观测性。  
夜晚，我把重复劳动写成工具，把经验沉淀成产品。  

我相信一种极客式修行：

> 少一点空谈，多一点可运行的代码；  
> 少一点“概念演示”，多一点“真实交付”。

这就是我的 `Zen AI Coding`：
- 以终为始：先定义真实场景，再选模型与架构。
- 小步快跑：先做 CLI，再做 API，再做生态。
- 持续复利：让每个脚本都能长成产品矩阵的一部分。

## 03. 公众号排版产品矩阵（API + 工具 + 插件）

### 核心能力
- 用 Markdown 写公众号文章，一键转换为精美排版并自动上传到微信草稿。
- 提供专业公众号排版 API，支持产品化接入与规模化生产。

### API 服务
- 国际站 API 文档: [md2wechat.app/api-docs](https://md2wechat.app/api-docs)
- 国内站 API 文档: [md2wechat.cn/api-docs](https://md2wechat.cn/api-docs)

### 生态项目
- 公众号排版 Skill: [md2wechat-skill](https://github.com/geekjourneyx/md2wechat-skill)
- Obsidian 插件: [obsidian-md2wechat](https://github.com/geekjourneyx/obsidian-md2wechat)
- 飞书转公众号浏览器插件: [feishu-md2wechat](https://github.com/geekjourneyx/feishu-md2wechat)
